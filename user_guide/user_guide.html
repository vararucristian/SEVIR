1. HOME PAGE

The Home page is also the first step of the generation server, where the user can chose where he will want to go.

2. INTERESTS PAGE

On this page the user can select his area of interests. He can choose between:

  • Museums
  • Shopping
  • Attraction
  • History
  • Relaxation
  • Tourism
  • Sports

3. ANY CHILD PAGE

On this page the user choose if he is traveling with children or not.

4. TIME PAGE

On this page the user have to choose how many hours he want to spend on stops.

5. MAP PAGE

On this page the user will be able to see the map with the generated route.

By clicking on a marker a pop up will be displayed with data from wikidata.

By clicking on Facebook/Pinterest button a new page will be open that will allow the user to share an image on Facebook/Pinterest.

The user can retake the survey by clincking on 'Search for other location'

6. LANGUAGE SELECTOR

The language can be changed by clicking on the drop down from the left top corner.

If the Romanian language is selected than all the text on the page is in Romanian.
